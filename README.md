GPACalci 🎓

GPACalci is a smart academic planning web application that enables students to calculate Semester GPA, track CGPA, predict future graduation outcomes using a What-If Simulator, and analyze academic performance through interactive dashboards. The application features a modern responsive UI, automatic calculations, local data storage, PDF/CSV export, and intuitive visualizations to help students plan and monitor their academic journey.

✨ Features
📚 Semester GPA Calculator
🎯 CGPA Calculator
🔮 What-If CGPA Predictor
📈 Performance Analytics Dashboard
📊 Interactive Charts (Chart.js)
💾 Local Storage Support
📄 PDF & CSV Report Export
🌙 Dark/Light Theme
